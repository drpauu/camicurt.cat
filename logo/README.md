# camicurt logo pack

Contingut recomanat per a una carpeta `/public/logo` o `/assets/logo` al repo.

## Fitxers inclosos
- `logo-master.png`: master transparent
- `logo-1024.png`, `logo-512.png`, `logo-256.png`, etc.
- `favicon.ico`
- `favicon-32x32.png`
- `favicon-16x16.png`
- `apple-touch-icon.png`
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`
- `maskable-192x192.png`
- `maskable-512x512.png`
- `site.webmanifest`
- `head-snippet.html`

## Ús recomanat
1. Mou aquesta carpeta a `public/logo`
2. Afegeix el contingut de `head-snippet.html` al `<head>` de la web
3. Usa `logo-512.png` o `logo-master.png` com a base per futures adaptacions

## Nota
Aquest pack surt d'un PNG transparent. No inclou un SVG vectorial natiu.
